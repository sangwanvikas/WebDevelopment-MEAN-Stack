JSON - JAVASCIPT OBJECT NOTATION

In JavaScript, data is represented using JSON or JavaScript Object Notation. JSON has become a very popular format for representing and sharing data in Web applications.

One object might have many different parts like Alica can have name, age, salary.

We have name:value pair of properties in the object. var alice ={name: 'abc', age:60, salary:5000}

Inmemory representation of JSON - JSON.stringify (object -> string) and JSON.parse (string -> object)
