Intro to javascript

JavaScript can be embedded in an HTML document by using the script element. Usually the script element goes in the head element, but it can appear anywhere in the HTML document and there can be any number of script elements. It is common to declare script elements at the end of the body element to improve SEO (Search Engine Optimization) performance, and page rendering performance. Within the script we can embed JavaScript code such as variables, arrays, objects, loops, branching, and functions.
