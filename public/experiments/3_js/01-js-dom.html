what is DOM ?

The browser downloads the document into its file cache and parses it into its memory. The document is parsed into an object representation of the document referred to as the DOM or Document Object Model. The DOM has an object representation of everything that is rendered on the page including the text, font, colors, layout, and white spaces in between. The DOM is the browser's interpretation of the plain text HTML document that is downloaded. Whereas the HTML might have syntax errors or missing elements, improper nesting, missing closing tags, the DOM is the interpretation of the text into a proper object representation of the page.

what is document in javascript ?

With JavaScript, it is possible to programmatically query, interact, and manipulate the DOM. Any modifications to the DOM are reflected onto the page. By changing the DOM, JavaScript can dynamically change what is displayed in the browser. The HTML page is available through JavaScript through an object called document. Through the document object we can use JavaScript to remove or add elements, change element's properties such as background color, font, padding and margins.
